# FEATURES — alezhu/dotenv-editor (fork)
Source: github.com/alezhu/dotenv-editor · v2.3.0 · MIT

Fork of koel/dotenv-editor (itself a fork of jackiedo). Diffs from jackiedo: (1) Illuminate constraint widened to `^5.8 … ^12`; (2) NEW end-of-line / final-newline control — `config.php` gains a `writer` block (`EOLMode` = os|windows|unix, `endsWithLinebreak` bool); `DotenvEditor` reads those config keys and pushes them into the writer on construct; `DotenvWriter` gains EOL-mode constants and honours them when joining the buffer (replaces the hardcoded `implode(PHP_EOL,…).PHP_EOL`); (3) minor whitespace normalization. Net-new public API: on `DotenvWriter` — `getEOLMode()`, `setEOLMode(string)`, `isEndsWithLinebreak()`, `setEndsWithLinebreak(bool)`. Tests: partial/misplaced (a `src/DotenvWriterTest.php` ships inside `src/` rather than a `tests/` dir; require-dev has phpunit ^12.4, mockery, faker, but no phpunit.xml). Abandoned/stale: reasonably current (tagged v2.3.0, Laravel up to 12, PHPUnit 12 dev dep) — the only fork adding genuine new functionality; minor hygiene smell (test file under src/, no test config).
